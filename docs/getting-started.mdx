# Getting Started

Welcome to Vail Love Hunt! This guide helps you run the app and the docs locally.

## Prerequisites

- Node.js 18+
- npm (this repo uses `package-lock.json`)

## Install and run

```bash
npm ci
npm run start
```

This starts the UI and API dev servers concurrently.

## Documentation site

```bash
npm run docs:dev
```

## Storybook

```bash
npm run sb:dev
```
